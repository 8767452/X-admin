**框架使用：**

环境要求：

> * node版本 10+
> * npm版本 6+

```javascript
1.首先git clone下来
2.使用脚手架目录打开cmd，输入命令npm install（如果失败请使用管理员权限打开cmd）
3.在命令行中输入npm start
```
